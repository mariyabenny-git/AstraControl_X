# AstraControl-X: Autonomous C2 Simulation Platform

**Project Entry for Hackverse X 2026** *Tracks: AI & Automation, Cybersecurity & Privacy, LLM with MCP*

## 🚀 The Mission
Modern defense systems face a **"Cognitive Load Crisis."** During high-velocity missile flight, tactical commanders are overwhelmed by raw telemetry, making it nearly impossible to detect and respond to complex electronic warfare (EW) attacks in real-time. **AstraControl-X** is a high-fidelity Command & Control (C2) simulation platform that acts as a force multiplier, utilizing AI to assist in tactical decision-making and mission integrity.

## 🛠 Core Innovation
Our platform bridges the gap between raw hardware data and tactical human intelligence:
* **Dynamic Telemetry Engine**: Simulates LRLACM flight physics (Speed/Altitude/Fuel) using a precise state-machine architecture for real-time performance.
* **Cyber-Resilience Framework**: A dedicated EW module that triggers when detecting simulated GPS spoofing or signal jamming, visualizing the immediate shift to secure frequency-hopping modes.
* **MLP-Heuristic Decision Support**: Unlike standard dashboards, our AI Strategic Advisor uses an MLP-based heuristic engine to process flight parameters. It calculates **Threat Probability Coefficients** to provide commanders with actionable, natural-language tactical recommendations.



## 🏗 System Architecture
The platform is built on a modular, low-latency framework designed for future integration with **Model Context Protocol (MCP)**, allowing Large Language Models to interface directly with live C2 data streams to provide advanced tactical insights.

| Module | Purpose |
| :--- | :--- |
| **Telemetry Grid** | Real-time sensor-fusion display (Mach, Altitude, Fuel levels). |
| **Cyber Shield** | Electronic Warfare detection and counter-measure simulation. |
| **AI Advisor** | Neural-heuristic engine for tactical threat assessment. |
| **Engine Diags** | Real-time waveform monitoring of propulsion health. |
